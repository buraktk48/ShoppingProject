Shopping / Product Tracking System (Python + SQLite)

A lightweight command-line application that enables product management, barcode-style product scanning, expiration date validation, and revenue tracking.

Features
- Add products (name, price, expiration date, manufacturer)
- Store products both in memory (dictionary) and in an SQLite database
- Scan products / build a shopping cart via CLI by entering product names
- Expiration date (expiry) check: prevents processing of expired product
- Query product count by manufacturer
- Search products by name
- Calculate total revenue

Alışveriş / Ürün Takip Sistemi (Python + SQLite)

Basit bir komut satırı uygulamasıyla ürün ekleme, ürün okutma, son kullanma tarihi kontrolü ve hasılat takibi yapan mini bir sistem.

Özellikler
- Ürün ekleme (ad, fiyat, SKT, üretici firma)
- Ürünleri hem sözlükte hem de SQLite veritabanında saklama
- Ürün okutma / sepet oluşturma (CLI üzerinden ürün adı girilerek)
- SKT kontrolü: son kullanma tarihi geçmiş ürünleri engeller
- Firmaya göre ürün sayısı sorgulama
- Ürüne göre arama 
- Toplam hasılat hesaplama
